 Two Types of Variables 
	 -Primitive 
	- Object Reference Variable: holds a way to access the object (something like a pointer/reference). JVM knows how to use references to get to the object 
- Instance Variable vs Local Variable 
	- IV are defined in the class while LV are defined within a method 
	- make IV private and use getter/setter method to set and access variable value.
	- LV must be initialised before being used. However, IV if not initialised has a default value.
