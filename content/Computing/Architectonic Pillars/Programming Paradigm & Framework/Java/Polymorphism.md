
#  Question on Polymorphism  
1. What is polymorphism in Java?
	1. It is the ability of a method to take different shape based on the context. 
	2. It is achieved through method overloading(compile time) and method overriding(runtime).
	3. Method overloading is method of same name but different parameter while method overriding with method with same name and same parameter in subclass but overridden by using @override tag.
