
Inheritance Tax, Reference:  [[Pragmatic Progammer]]
1. Fragile Base Class Problem - changes in base class impacts all classes that inherited it. 
2. Unnecessary complexity arising out of multiple level of inheritance 
3. Avoid inheritance for code reuse, better to use [[Composition]]
4. Violates [[Encapsulation]] as subclass depends on working of parents class making changes to parents risky.
