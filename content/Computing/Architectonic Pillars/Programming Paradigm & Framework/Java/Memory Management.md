1. **Garbage Collection** — Abandoned or unreferenced objects are eligible for GC. The JVM automatically reclaims heap memory when objects become unreachable.
2. Heap and Stacks are Two Memory Space created by JVM 
3. Object and Instance Variables lies in Heap while Local Variable, reference of object within the method and method lies in Stack.
