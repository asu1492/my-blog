Data Types in Java 
1. Reference Type
	1. Interfaces including annotations 
	2. Classes including enums 
	3. Arrays
2. Primitives 
- Class instances and Arrays are objects but primitives are not.  
- Method signature consists of the method **name** and type of **parameters** and not the return type of method. 
- A class implements an interface, this is nothing but inheritance. 
- API refers to classes, interfaces, constructors, members, and serialized forms by which a programmer accesses a class, interface or package. 
- classes, interfaces, constructors, members, and serialized forms are collectively known as API elements. 
- Exported API of a package consists of the public and protected members and constructors of every public class or interface in the package.
