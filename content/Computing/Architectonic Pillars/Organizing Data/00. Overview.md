## Interview Quick-Reference

- **Array/String problems** → start with [[Two Pointers and Sliding Window]]
- **Graph problems** → start with [[Graph, Trees and Heap]] (BFS/DFS decision tree)
- **Optimisation problems** → check [[Dynamic Programming]] state definition first
- **Sorted input** → consider [[Binary Search + Greedy]]
- **Range/overlap problems** → [[Interval]]
