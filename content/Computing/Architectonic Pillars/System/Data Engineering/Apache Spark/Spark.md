What is Spark 

1. Spark is an open source **in memory** application framework for **DDP(distributed data processing)** and iterative analysis on massive data volumes. 
2. Written in Scala and runs on JVM
3. Runs on clusters based on linux os.

Why Spark 

1. 100X more faster than MapReduce in memory or 10x faster on disk. 
2. Ease of Use : can write application in Java, Scala, Python, R 
3. Generality : combine SQL, streaming and complex analytics -- powers a stack of libraries including SQL and DataFrames, MLib for Machine Learning, Graphx and Spark Streaming 
4. Provides parallel and distributed processing
5. Scalability and Modularity
6. Fault tolerances and redundancy
7. Solves read/write problems of mapreduce through in memory
