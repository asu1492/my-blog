Replication 
1. Fragmented data is stored redundantly in two or more sites
2. increases avl of the data 
3. is carried out to protect data from node failure
4. Replicated data need to be synchronized, could lead to inconsistencies 
5. can be High Availability Replica - exist at same location ; prone to site based disaster. Thus replicas are setup in geographically distributed locations. ![[image (10).png]]
