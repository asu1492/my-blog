What is db 

1. it is a repository of data, provides functionality for adding, modifying and querying that data. 
2. Diff kinds f db store data in diff forms. 
3. terms such as db, db server, db system, data server, dbms are used interchangeably. 

1. MongoDB (boat)
2. PostgreSQL (elephants)
3. Neo4j (hot air baloon)
4. Redis (car)

* Job requirement determines which vehicles you choose, same with databases. 

* * *

Schema 

1. If database is a table in Excel, schema is the columns. 
2. Some databases have strict schema like PostgreSQL while other like MongoDB invent it on fly. 

_**Types of Databases**_

1. Relational Databases
	1. data is stored in table format -- rows and column 
	2. Table is collection of related things. 
	3. In RDB, can form relationship b/w tables. 
2. Non Relational 
	1. Document based databases , e.g. NoSQL 
	2. A Graph based DB, 
	3. Key - Value 
	4. Wide Column 
		1. Google Bigtable 
		2. Apache Cassandra, (used by FB to store its data) 
		3. known for their scalability & resilience 
3. Others
	1. Search Engines : like solr, sphinx
	2. Message brokers, e.g. Kafka, RabbitMQ 
	3. Multi Model DB 
		1. fulfill more than one of the above paradigm 
		2. Azure Cosmos DB, ArangoDB are primary e.g. 
		3. MongoDB and PostgreSQL also technically qualify as MMDB with their ability to bend into different shapes
