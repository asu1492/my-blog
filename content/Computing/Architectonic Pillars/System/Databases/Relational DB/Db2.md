Various Db2 Product

1. Db2 Warehouse -- massive parallel processing(MPP) and ML ; make use of data nodes to store data, thus can be easily scaled by adding data nodes. 
2. Db2 on Cloud -- provides performance, high avl, scalability and resilience 
3. Db2 Big SQL -- SQL on Hadoop provides MPP, advanced querying, works with Hadoop HDFS and WebHDFS, RDBMS, NoSQL. Can be integrated with Cloudera Data Platform and IBM Cloud Pak for Data 
4. Db2 Event store -- memory optimized, used for streaming data for event driven applications, includes IBM Watson Studio 
5. Db2 for z/OS - for mission critical and security features 
6. Cloud Pak for Data -- fully integrated data and AI Platform, Runs on Red Hat Openshift in a container, thus can be deployed in any private or public or hybrid cloud.
