# Abhishek Singh — Digital Garden

Personal notes, blog, and knowledge management site built with [Quartz 4](https://quartz.jzhao.xyz/).

## Setup

Requires Node v22+ and npm v10.9.2+.

```bash
nvm use 22
npm install
```

## Local Preview

```bash
npx quartz build --serve
```

Then open [http://localhost:8080](http://localhost:8080).
